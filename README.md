# EXPIPs [![Gitter](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/expanse-org/EXPIPs?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge)
Expanse Improvement Proposals (EXPIPs) describe standards for the Expanse platform, including core protocol specifications, client APIs, and contract standards.

# Contributing
First review [EXPIP-1](EXPIPS/EXPIP-1.md). Then clone the repository and add your EXPIP to it. There is a [template EXPIP here](EXPIP-X.md). Then submit a Pull Request to Expanse's [EXPIPs repository](https://github.com/expanse-org/EXPIPs).

# EXPIP status terms
* **Draft** - an EXPIP that is open for consideration
* **Accepted** - an EXPIP that is planned for immediate adoption, i.e. expected to be included in the next hard fork (for Core/Consensus layer EXPIPs).
* **Final** - an EXPIP that has been adopted in a previous hard fork (for Core/Consensus layer EXPIPs).
* **Deferred** - an EXPIP that is not being considered for immediate adoption. May be reconsidered in the future for a subsequent hard fork.
